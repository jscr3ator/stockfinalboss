# StockFinalBoss
> Check out the public demo [here](https://stockfinalboss.jscreator.dev/)

**Simulate stocks with others!**

This project is a multiplayer stock market simulation that uses web servers to connect players. 

It was created for the **Inspirit AI workshop** through vibe coding tools around sub 30 mins!
